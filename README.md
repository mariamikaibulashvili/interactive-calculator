# interactive-calculator

A clean, modern, and secure interactive calculator built using semantic HTML5, CSS Flexbox, and Vanilla JavaScript. 

## Features
- **Modern UI/UX:** Dark mode design with smooth hover and click (active) animations.
- **Secure Architecture:** Avoids the unsafe usage of `eval()`, implementing custom state-management for mathematical operations instead.
- **Error Handling:** Built-in protection against edge cases, such as division by zero.

## Technologies Used
- HTML5
- CSS3 (Flexbox & Pseudo-classes)
- JavaScript (DOM Manipulation & State Management)
